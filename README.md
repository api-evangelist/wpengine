# WP Engine (wpengine)

WP Engine is a WordPress digital experience platform offering managed WordPress hosting with a REST API for programmatically managing sites, environments (installs), domains, SSH keys, SSL certificates, backups, and cache operations. The API uses Basic Authentication with credentials generated from the WP Engine User Portal.

## APIs

### WP Engine Hosting Platform API

REST API for programmatically managing WP Engine WordPress hosting accounts, sites, installs (environments), domains, SSH keys, SSL certificates, backups, users, and cache operations.

- **Human URL:** [https://developers.wpengine.com/docs/managed-hosting-platform/api](https://developers.wpengine.com/docs/managed-hosting-platform/api)
- **Base URL:** `https://api.wpengineapi.com/v1`
